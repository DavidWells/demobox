Demobox has 19 color palettes (from Google's material ui) and 18 font pairings (from femmebot's google type project). These can be used in any combination. The following screenshots display these colors and fonts in example combinations.

The "head" font is the one used to identify a font pair. For example, to get the first example, you would have in your header:

colors: red
fontPair: Alfa Slab One
